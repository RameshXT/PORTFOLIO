Git
The Version Control System
My Git Knowledge Level
★ ★ ★ ★ ★
Git is a distributed version control system for tracking changes in source code during software development.

Key Features:

Commit: Record changes to the repository.

Branch: Create separate lines of development.

Merge: Combine changes from different branches.

Pull Request: Propose changes to a repository.

Rebase: Reapply commits on top of another base tip.

Stash: Temporarily save changes without committing.

Popular Use Cases:

Version control, Collaborative development, Code review

Kubernetes
Automating Deployment, Scaling, and Management
My Kubernetes Knowledge Level
★ ★ ★ ★ ★
Kubernetes is an open-source system for automating the deployment, scaling, and management of containerized applications. It helps manage containerized workloads and services, facilitating declarative configuration and automation.

Key Features:

Container Orchestration: Automates deployment, scaling, and operations of containers.

Pods: The smallest deployable units in Kubernetes, containing one or more containers.

Auto-Scaling: Dynamically adjusts the number of running containers based on load.

Service Discovery: Automatically detects services and ensures high availability.

ConfigMaps: Manages configuration data separately from application code.

Secrets: Securely stores sensitive information like passwords and tokens.

Popular Use Cases:

Managing microservices, automated deployments, and container orchestration.

Explore Kubernetes
Terraform
Infrastructure as Code
My Terraform Knowledge Level
★ ★ ★ ★ ★
Terraform is an open-source tool that allows you to define and provision infrastructure using a high-level configuration language.

Key Features:

Plan: Preview changes before applying them to your infrastructure.

Apply: Execute the changes and update your infrastructure.

Validate: Ensure configuration files are error-free.

State: Manage and track the state of your infrastructure.

Modules: Create reusable configurations to manage resources.

Output: Capture and display key information from your configurations.

Popular Use Cases:

Provisioning cloud infrastructure, Managing multi-cloud environments, Automating infrastructure deployment

NGINX
The Engine X of the Internet
My NGINX Knowledge Level
★ ★ ★ ★ ★
NGINX is a high-performance HTTP server and reverse proxy, as well as an IMAP/POP3 proxy server. It powers many of the world's most popular websites.

Key Features:

HTTP Server: Handles web requests with efficiency.

Load Balancer: Distributes traffic to different servers.

Reverse Proxy: Directs client requests to the appropriate server.

SSL Termination: Offloads SSL decryption to improve performance.

Caching: Speeds up web content delivery by storing responses.

Static Files: Efficiently serves static content like HTML, CSS, and images.

Popular Use Cases:

Web server, Load balancing, API gateway

Explore NGINX

Running Java-based web applications, Servlet container, JSP engine
